🎓 Pregrado - Drexel University

Triple especialización en Finanzas, Economía y Análisis de Datos

💰 Finanzas (Finance)

¿Qué es?

Las finanzas son el estudio y gestión del dinero, inversiones y otros instrumentos financieros. Se enfoca en cómo las personas, empresas y organizaciones asignan recursos a lo largo del tiempo, considerando los riesgos y oportunidades del mercado. Las finanzas corporativas ayudan a las empresas a tomar decisiones sobre inversiones, estructura de capital y gestión de activos.

Herramientas Tecnológicas

  • Excel / VBA: Modelado financiero, análisis de inversiones, valuación de empresas
  • Bloomberg Terminal: Datos de mercado en tiempo real, análisis de bonos y acciones
  • Python: Análisis cuantitativo, automatización de reportes, backtesting de estrategias
  • SQL: Consultas de bases de datos financieras, extracción de datos históricos
  • Tableau / Power BI: Visualización de métricas financieras y KPIs
  • FactSet / Capital IQ: Investigación financiera y análisis fundamental

Tipos de Trabajos

  • Analista Financiero: Evaluación de inversiones y preparación de informes financieros
  • Analista de Inversiones: Análisis de portafolios, acciones y bonos
  • Banca de Inversión: Fusiones, adquisiciones, emisión de valores
  • Gestor de Portafolios: Administración de fondos de inversión
  • Planificador Financiero: Asesoría financiera personal y patrimonial
  • Analista de Riesgos: Evaluación y mitigación de riesgos financieros

Aptitudes Clave para el Éxito

  • Pensamiento analítico y habilidades cuantitativas
  • Comprensión profunda de mercados y economía
  • Atención al detalle y precisión
  • Capacidad de tomar decisiones bajo presión
  • Comunicación efectiva de información financiera compleja
  • Ética profesional y integridad

Rango Salarial

$60K - $90K Estados Unidos (Inicial)
S/50K - S/80K Perú (Inicial)

Nota: Los salarios pueden aumentar significativamente con experiencia. Senior y directivos pueden superar $150K+ en EEUU y S/150K+ en Perú.

📈 Economía (Economics)

¿Qué es?

La economía estudia cómo las sociedades utilizan recursos limitados para producir bienes y servicios, y cómo los distribuyen entre diferentes grupos. Analiza el comportamiento de mercados, políticas gubernamentales, comercio internacional y desarrollo económico. Los economistas utilizan modelos matemáticos y datos empíricos para comprender fenómenos económicos y predecir tendencias.

Herramientas Tecnológicas

  • R: Análisis estadístico, econometría, modelado de series temporales
  • Stata: Software especializado en econometría y análisis de datos panel
  • Python: Análisis de políticas, machine learning aplicado a economía
  • MATLAB: Modelado económico, simulaciones y optimización
  • EViews: Forecasting económico y análisis de series de tiempo
  • SPSS: Análisis estadístico de encuestas y datos socioeconómicos

Tipos de Trabajos

  • Economista de Investigación: Análisis de políticas públicas y tendencias económicas
  • Analista Económico: Evaluación de impacto económico para empresas o gobierno
  • Consultor Económico: Asesoría a empresas en estrategias de mercado
  • Economista de Banco Central: Política monetaria y análisis macroeconómico
  • Analista de Mercado: Investigación de tendencias de consumo y competencia
  • Economista Internacional: Comercio exterior, desarrollo económico global

Aptitudes Clave para el Éxito

  • Sólida base en matemáticas y estadística
  • Pensamiento crítico y capacidad analítica
  • Comprensión de políticas públicas y sociales
  • Habilidad para interpretar grandes volúmenes de datos
  • Comunicación clara de conceptos complejos
  • Curiosidad intelectual y actualización constante

Rango Salarial

$55K - $85K Estados Unidos (Inicial)
S/45K - S/75K Perú (Inicial)

Economistas senior en organismos internacionales o consultoras top pueden ganar significativamente más.

📊 Análisis de Datos (Data Analytics)

¿Qué es?

El análisis de datos es el proceso de inspeccionar, limpiar, transformar y modelar datos para descubrir información útil, llegar a conclusiones y apoyar la toma de decisiones. Los analistas de datos convierten datos crudos en insights accionables mediante visualizaciones, reportes y análisis estadísticos, ayudando a las organizaciones a optimizar operaciones y estrategias.

Herramientas Tecnológicas

  • SQL: Extracción y manipulación de datos de bases relacionales
  • Python (Pandas, NumPy): Limpieza, transformación y análisis de datos
  • Tableau: Visualización interactiva y dashboards empresariales
  • Power BI: Business intelligence y reportería automatizada
  • Excel Avanzado: Tablas dinámicas, funciones complejas, macros
  • Google Analytics: Análisis de datos web y comportamiento de usuarios
  • R (ggplot2, dplyr): Análisis estadístico y visualización

Tipos de Trabajos

  • Analista de Datos: Análisis exploratorio, reportes y visualizaciones
  • Analista de Business Intelligence: Diseño de dashboards y KPIs
  • Analista de Marketing: Análisis de campañas, segmentación de clientes
  • Analista de Operaciones: Optimización de procesos y eficiencia
  • Analista de Producto: Análisis de uso y comportamiento de usuarios
  • Analista Financiero de Datos: Métricas financieras y forecasting

Aptitudes Clave para el Éxito

  • Dominio de herramientas de análisis y visualización
  • Pensamiento lógico y resolución de problemas
  • Atención al detalle y precisión en el trabajo
  • Capacidad de storytelling con datos
  • Conocimiento del negocio y contexto empresarial
  • Colaboración efectiva con equipos multidisciplinarios

Rango Salarial

$65K - $95K Estados Unidos (Inicial)
S/55K - S/85K Perú (Inicial)

Campo en alta demanda. Analistas senior pueden superar $120K en EEUU y S/100K en Perú.

🤖 Ciencia de Datos (Data Science)

¿Qué es?

La ciencia de datos es un campo interdisciplinario que utiliza métodos científicos, algoritmos, procesos y sistemas para extraer conocimiento e insights de datos estructurados y no estructurados. Combina estadística, informática, machine learning y conocimiento del dominio para resolver problemas complejos, hacer predicciones y automatizar decisiones. Los científicos de datos construyen modelos predictivos, sistemas de recomendación y soluciones de inteligencia artificial.

Herramientas Tecnológicas

  • Python (Scikit-learn, TensorFlow, PyTorch): Machine learning y deep learning
  • R: Análisis estadístico avanzado y modelado predictivo
  • SQL / NoSQL: Gestión de bases de datos relacionales y no relacionales
  • Apache Spark: Procesamiento de big data distribuido
  • AWS / Azure / GCP: Infraestructura cloud y MLOps
  • Jupyter / VS Code: Entornos de desarrollo y documentación
  • Git: Control de versiones y colaboración
  • Docker / Kubernetes: Contenedorización y deployment de modelos

Tipos de Trabajos

  • Científico de Datos: Desarrollo de modelos predictivos y análisis avanzado
  • Machine Learning Engineer: Implementación y deployment de modelos ML
  • Ingeniero de Datos: Arquitectura de pipelines ETL y data warehousing
  • AI Researcher: Investigación en algoritmos de inteligencia artificial
  • MLOps Engineer: Automatización y monitoreo de sistemas ML
  • Data Science Consultant: Asesoría estratégica en proyectos de datos

Aptitudes Clave para el Éxito

  • Fuerte base matemática (álgebra lineal, cálculo, probabilidad)
  • Programación avanzada y pensamiento algorítmico
  • Conocimiento profundo de machine learning y estadística
  • Capacidad para trabajar con grandes volúmenes de datos
  • Creatividad para resolver problemas no estructurados
  • Comunicación efectiva de resultados técnicos
  • Aprendizaje continuo (campo en constante evolución)

Rango Salarial

$90K - $140K Estados Unidos (Inicial)
S/80K - S/120K Perú (Inicial)

Una de las profesiones mejor pagadas en tecnología. Senior Data Scientists y ML Engineers pueden superar $200K+ en EEUU (especialmente en empresas FAANG) y S/150K+ en Perú.
